# My personal dotfiles and scripts

## Installation
```
git clone https://github.com/einzigartigerName/dotdotdot
cd dotdotdot
./install [option]
```
### Usage
```
Usage:
    -c  install configs
    -s  install scripts
    -d  install dependencies
    -a  install all
```
